## 개요

- 파이썬 패키지인 networkx를 이용해서 소셜네트워크를 시각화하는 코드입니다.
- [networkx tutorial](https://networkx.github.io/documentation/stable/index.html) 

## 순서

- 데이터 로드
- 네트워크 시각화
- 중심성 계산하기
- 서브그래프 생성하기
- Appendix
  - 랜덤 그래프 생성하기
  - 노드/엣지 추가하기
  - 노드/엣지 속성 반영하기
